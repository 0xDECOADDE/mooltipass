41 /**************************************************
42 * SSD1322 driver for 256x16x4 OLED display
43 *
44 * Pin Function SPI connections
45 * ----+------------+----------
46 * 1 VSS GND
47 * 2 VBAT 3.3V-5V
48 * 3 NC
49 * 4 D0 SCLK Serial Clock
50 * 5 D1 MOSI Serial Data Input
51 * 6 D2 NC
52 * 7 D3 GND
53 * 8 D4 GND
54 * 9 D5 GND
55 * 10 D6 GND
56 * 11 D7 GND
57 * 12 #RD GND
58 * 13 #WR GND
59 * 14 DC Data / Command
60 * 15 #RESET
61 * 16 #CS Chip select
62 *
63 * Note: All logic pins are 3.3V max.
64 *
65 * BS1 BS0 Mode
66 * --------+---------------------
67 * 0 0 "4 Line SPI" 8-bit + DC pin
68 * 0 1 "3 Line SPI" 9-bit DC is 9th bit
69 * 1 0 8-bit 8080 parallel
70 * 1 1 8-bit 6800 parallel
71 *
72 * Note: SPI mode is write only (MOSI)
73 *
74 **************************************************/
