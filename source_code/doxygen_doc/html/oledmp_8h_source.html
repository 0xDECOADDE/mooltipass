42 /**************************************************
43 * SSD1322 driver for 256x16x4 OLED display
44 *
45 * Pin Function SPI connections
46 * ----+------------+----------
47 * 1 VSS GND
48 * 2 VBAT 3.3V-5V
49 * 3 NC
50 * 4 D0 SCLK Serial Clock
51 * 5 D1 MOSI Serial Data Input
52 * 6 D2 NC
53 * 7 D3 GND
54 * 8 D4 GND
55 * 9 D5 GND
56 * 10 D6 GND
57 * 11 D7 GND
58 * 12 #RD GND
59 * 13 #WR GND
60 * 14 DC Data / Command
61 * 15 #RESET
62 * 16 #CS Chip select
63 *
64 * Note: All logic pins are 3.3V max.
65 *
66 * BS1 BS0 Mode
67 * --------+---------------------
68 * 0 0 "4 Line SPI" 8-bit + DC pin
69 * 0 1 "3 Line SPI" 9-bit DC is 9th bit
70 * 1 0 8-bit 8080 parallel
71 * 1 1 8-bit 6800 parallel
72 *
73 * Note: SPI mode is write only (MOSI)
74 *
75 **************************************************/
