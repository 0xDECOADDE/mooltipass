39 /**************************************************
40 * LM320Y-256064 (SSD1322 driver)
41 *
42 * Pin Function SPI connections
43 * ----+------------+----------
44 * 1 VSS GND
45 * 2 VBAT 3.3V-5V
46 * 3 NC
47 * 4 D0 SCLK Serial Clock
48 * 5 D1 MOSI Serial Data Input
49 * 6 D2 NC
50 * 7 D3 GND
51 * 8 D4 GND
52 * 9 D5 GND
53 * 10 D6 GND
54 * 11 D7 GND
55 * 12 #RD GND
56 * 13 #WR GND
57 * 14 DC Data / Command
58 * 15 #RESET
59 * 16 #CS Chip select
60 *
61 * Note: All logic pins are 3.3V max.
62 *
63 * BS1 BS0 Mode
64 * --------+---------------------
65 * 0 0 "4 Line SPI" 8-bit + DC pin
66 * 0 1 "3 Line SPI" 9-bit DC is 9th bit
67 * 1 0 8-bit 8080 parallel
68 * 1 1 8-bit 6800 parallel
69 *
70 * Note: SPI mode is write only (MOSI)
71 *
72 **************************************************/
